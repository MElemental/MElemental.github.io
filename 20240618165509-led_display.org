#+title: LED display
#+filetags: :LED:video:playback:
LED Displays use an array of addressable [[id:c6e9553e-6262-484a-bc1e-baafdcf4044b][LEDs]] to playback video.
Each LED corresponds to a pixel on a display.
LED Displays usually exist in convention settings, outdoor signage, and live concerts.
